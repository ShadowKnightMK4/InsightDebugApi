◆ DebugExceptionInPage_Type

enum InsightSheath.Debugging.DebugExceptionInPage_Type

Type of In Page exception flags. You can check if the exception is that at DebugEventExceptionInfo.ExceptionCode

This file provides a wrapper between c# and the InsightApi.Dll that deals with the debug event structure. The base class DebugEventStaticContainer is the root for the other classes in this file. It handles holding an IntPtr that represents the unmanaged DEBUG_EVENT structure. This pointer gets passes to various routines in the InsightApi.Dll DLL which then return data bases on that pointer - usually about the DEBUG_EVENT struct. Each DEBUG_EVENT in the MSDN documentation has its own C# class DebugEventType About Exceptions The MSDN documentation has certain exceptions listed. Those have been added to the DebugExceptionTypes enum. That's not actually exhaustive. winbase.h lists MANY MANY MANY more.

Target process thread with the DebugExceptionTypes.InPageError tried writing to something it's not allowed too
