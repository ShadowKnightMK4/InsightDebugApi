readonly MachineType InsightSheath.Telemetry.GeneralTelemtryHelperStruct.Type

Depends on what DebugEvent.IsProcess32Bit returns. Intended for determining pointer size only. If that's true, Value is MachineType.MachineI386 otherwise value is MachineType.MachineAmd64 i386 pointer sizes are 4 bytes long and Amd64 pointer sizes are 8 bytes. This may not actually be the exact machine type as stored in the debugged process's physical PE/EXE file.
