override void InsightSheath.Win32Struct.WindowsObjectAttributes.Blit ( )
inlineprotectedvirtual

This code should marshal the relevant struct from the native pointer if WasBlit is not set and set afterwards. Accessors should call this each time they are called to retrieve values. Additionally, if the .NET struct will need to make changes to the land, you're going to need to call Apply BEFORE reading the managed struct. Not doing this will revert your changes on the managed side to the equivalent native values.
