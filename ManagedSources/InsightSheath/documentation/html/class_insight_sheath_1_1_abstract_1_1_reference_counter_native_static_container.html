Detailed Description

This class contains an AddRef()/ Release() component for child classes. Destructors(), Call Release() and if that returns 0, free as appropriate. Not needed if the class itself is a 1 to 1 ie Sheath class pointer that contains one pointer, this can skip. Sheath, DebugEvent classes have this in the change, because they give out multiple references to a native class like candy.
