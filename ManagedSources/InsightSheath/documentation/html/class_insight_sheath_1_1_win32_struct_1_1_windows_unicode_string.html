override void InsightSheath.Win32Struct.WindowsUnicodeString.Dispose ( bool  disposing)
inlineprotectedvirtual

The native memory is either a UnicodeString32 or UnicodeString64 struct Dependant on StructType. Pointer is a separate allocated string. This means NativeImports.NativeMethods.SimpleFree(IntPtr) is incorrect in this scenario. We'll need to call NativeImports.NativeMethods.RemoteFreeUnicodeString(IntPtr, bool) pointer here. Also FreeOnCleanup should be false if this structure is part of a larger struct directly and NOT A POINTER.
